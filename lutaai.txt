汝塔v1.7听频道生图参考提示词
汝塔｜六度播客封面生图规则（可直接交付设计师）v1.0
1.1 交付规格（固定）
•输出类型：播客封面
•画幅：1:1 正方形
•分辨率：2048×2048（最低 1024×1024）
•安全边距：四周留白 10%（主体元素不得触边）
•元素数量上限（硬约束）
￮主要几何形体：≤ 4 个
￮线条：≤ 3 条
￮渐变/光晕：≤ 2 处（且只作用在形体内部或边缘，不做满屏背景渐变）
•颜色数量上限（硬约束）
￮彩色（有色相）≤ 2 种
￮中性（纸白/雾灰/极浅暖灰）≤ 1 种
￮合计 ≤ 3 色系（同一色相的明暗变化不计为新增色相）
•禁止文字与符号：任何文字、数字、标点、logo、字母、经文、符号一律禁止
•禁止具象：任何可识别物体、宗教器物、自然景物、人物/动物、器官/手势、建筑一律禁止
•交付文件
a.最终导出 PNG（已应用蒙版效果）
b.可编辑源文件（Figma/PSD/AI 任选其一）：必须包含“Base 图层 + Mask 组（可调参数）”

1.1 总体美学基调（日本美学融入，固定）
目标观感：清澈、轻松、不杂乱，并带有“静中有深”的余韵。
必须贯彻的日本美学特质（写入审美约束）
•間（Ma）：留白优先，主体信息密度低
•簡素（Kanso）：极简，不装饰，不堆叠细节
•渋い（Shibui）：克制、高级、不过分艳丽
•幽玄（Yūgen）：含蓄的深度（用微弱层次、半透明叠加表达）
•不均齐（Fukinsei）：轻微不对称（除“禅定”可允许对称）
•静寂（Seijaku）：画面干净、呼吸感强

1.2 颜色系统（从“六度”出发，并参考“五方佛五色”，不固定核心色）
2.1 五色参考（仅作为色相母库）
•蓝（东方）：澄明、冷静、深邃
•黄（南方）：温暖、开阔、滋养
•红（西方）：力量、温热、精进
•绿（北方）：生机、平衡、流动
•白（中央）：清净、澄澈、空灵（允许“纸白/雾白”，避免纯白刺眼）
注：颜色不“固定核心色”，但每张必须从上述五色中选择 1 个主色相 + 1 个对比色相（可选），并做“高明度、低杂讯”的清澈处理。
2.2 “清澈轻松”用色硬规则
•明度优先：主色相必须为“偏浅/偏透”的高明度版本（避免深沉重色块）
•饱和度克制：中低饱和为主；禁止荧光/霓虹
•对比方式限定：只允许用“明度对比 / 冷暖对比 / 面积对比”实现张力；禁止彩虹撞色
•背景建议（不固定某一个，但必须在此范围内）：
￮纸白/雾白/极浅暖灰/极浅米白（如：#F6F4EE、#F3F5F6、#F7F2E8 等同级别明度）
￮禁止纯黑背景作为常态（除非某一轮试产明确需要）
•每张最多 2 个有色相 + 1 个中性色（见 0 章硬约束）

1.3 反向排除库（Negative Library，扩展版，必须全程启用）
目的：强力对冲“过于具象、符号化、元素过多、花哨杂乱、宗教图标化”。
3.1 禁止具象与宗教物件
•佛像、菩萨像、人物、头像、手、眼睛、动物
•莲花、法轮、曼荼罗、宝伞、宝幢、法螺、金刚杵、铃杵、念珠、香炉、供灯、寺庙、塔、经卷、木鱼、坐垫、袈裟纹样
•山水、云、水波、火焰、日月星辰、树叶花草、石头、鸟兽
3.2 禁止“符号化元素 / 图案化元素”（重点扩展）
•任何文字系统：中文/日文/英文/梵文/藏文/符号字符
•任何“可读符号”：☸、卍/万字、Om、十字、星形符号、阴阳、八卦、六芒星、符咒、印章、书法笔触字形
•Enso 圆相（禅圈）、印章风朱文、篆刻感图形、经文条纹
•“神圣几何/圣几何”类：花之生命、密集同心花瓣、放射对称图腾、过密网格/伊斯兰几何纹
•UI/信息符号：二维码、条形码、波形图、播放键、音符、麦克风、耳机、按钮图标、任何界面元素
•叙事符号：箭头、对勾、X、感叹号、问号、定位针、罗盘、时钟等
3.3 禁止风格与质感
•摄影、写实 3D、金属写实、玻璃写实渲染（允许“抽象玻璃感”，但不写实）
•赛博霓虹、故障风、蒸汽波、强颗粒海报噪点、重纹理纸张纤维（如明显可见则禁）
•复杂拼贴、多层叠加导致“脏、乱、拥挤”的效果
•具有明显中式、新中式、中式山水、禅意、禅的印象。

1.4 快检机制（加强版：对冲“颜色过多/线条过多/过具象”）
每张图必须通过以下快检（任何一条不过即淘汰）：
4.1 结构快检（复杂度）
• 主要形体 ≤ 4
• 线条 ≤ 3
• 渐变/光晕 ≤ 2，且不覆盖满屏
• 画面留白 ≥ 50%（视觉占比）
4.2 颜色快检（清澈度）
• 有色相 ≤ 2，中性色 ≤ 1
• 主色为高明度版本（整体感觉轻）
• 无荧光/霓虹感，无“彩虹堆叠”
• 缩略到 128px 仍清爽、不糊成一团
4.3 抽象快检（去具象）
• 5 秒内看不出任何具体物体/宗教器物/自然景物
• 不出现“莲花/法轮/曼荼罗/禅圈/经文/印章”等符号化联想
• 无放射对称图腾（容易曼荼罗化）
4.4 品质快检（日本美学）
• “間（Ma）”成立：空间有呼吸
• “渋い（Shibui）”成立：高级克制，不浮夸
• “幽玄（Yūgen）”成立：有轻微层次但不堆细节

1.5 蒙版系统（Mask System）：每一度一套“可调蒙版规则”
通用要求
•蒙版必须是独立可编辑层组（Mask Group），包含：
a.Color Mask（渐变映射/叠色）
b.Opacity Mask（透明度控制）
c.Blur Mask（局部柔化，避免硬边噪声）
•最终导出必须为“已应用蒙版后的效果”，但源文件保留所有蒙版参数可调。
通用参数范围（建议）
•Color Mask：叠色不超过 12–22% 不透明度
•Opacity Mask：局部透明度变化幅度不超过 15%（避免脏）
•Blur Mask：20–80px（视分辨率调整），只用于“边缘过硬/层次过强”的部位
•禁止：颗粒噪点作为“质感补救”（容易脏乱）

1.6 批量生产规则（每一度一次生成 30 张）
•每一度生成 30 张，共 6×30=180 张
•每一度内部统一：构图语法与蒙版逻辑一致；但允许在“几何母题变体/色相选择/布局偏移”上变化
•建议分三轮出图：
￮Round A：10 张（偏保守、最干净）
￮Round B：10 张（加入轻微层次/半透明叠加）
￮Round C：10 张（加入更明确的对比，但仍满足颜色与形体上限）

7. 六度生图基础规则（含：构图语法 + 用色逻辑 + 蒙版规则 + 30 张变体方式）
说明：每一度给出“母题语法（固定） + 10 个几何变体家族”。生成 30 张时：每个变体家族出 3 张（换色相组合与布局微调即可），合计 10×3=30。

7.1 布施（Dāna）—“开放、流动、轻盈”
构图语法（固定）
•结构：开放的弧/环 + 向外舒展的流线（但不做箭头）
•空间：主体偏中心或偏一侧，留白更大
•复杂度：形体不超过 3 个，线条不超过 2 条
用色逻辑（从五色选）
•倾向表达“轻与暖”：主色优先 黄 / 绿 的高明度版本
•对比色可选 蓝 / 红（小面积点醒）
•白色只作为背景纸白或极浅高光，不做硬白块
蒙版规则（布施专属）
•Color Mask：径向渐变（中心→外缘），让色彩“向外扩散”
￮叠色 12–18%，边缘更淡
•Opacity Mask：外缘透明度略高（更轻），中心略实
￮变化幅度 ≤ 12%
•Blur Mask：对外扩边缘做柔化 30–60px，形成“气息感”
10 个几何变体家族（用于 30 张）
1.单一开放弧（缺口明显）
2.双弧交错但不闭合
3.三段短弧围合“未完成的环”
4.弧 + 一条轻薄流线外引
5.弧 + 半透明薄片向外漂移
6.两个小型开放环错位
7.软角矩形开口（非圆）
8.轻薄带状曲面（不写实）
9.点状光斑极少量外散（≤ 5 个点）
10.单块几何 + 边缘渐隐开口

7.2 持戒（Śīla）—“边界、秩序、澄净”
构图语法（固定）
•结构：框/界线/分区（但禁止密集网格）
•排列：对齐感强，间距规整
•复杂度：形体 ≤ 4，线条 ≤ 3
用色逻辑（从五色选）
•倾向“清净与克制”：主色优先 白 / 蓝 / 绿 的高明度版本
•对比色选 红 或 黄（极小面积，用于“警醒”但不躁）
蒙版规则（持戒专属）
•Color Mask：矩形边界式渐变（内清外收），形成“界线感”
￮叠色 10–16%
•Opacity Mask：边界处略实，内部略透（象征戒相）
￮变化幅度 ≤ 10%
•Blur Mask：只对“边界过硬”轻柔化 20–40px，保持清晰为主
10 个几何变体家族
1.单框（细线框）+ 内部留白
2.双框（框中框）
3.纵向分区（两域）但不做“切割锐利”
4.横向分区（两域）
5.细线边界与一块半透明面片
6.规则间距的 3 条平行线（≤3）
7.角落定位式几何（四角暗示边界）
8.软角框 + 内部浮动小块（≤1）
9.细线框 + 轻微偏移阴影面（非写实）
10.一条界线 + 两侧极简色域对比（面积对比）

7.3 忍辱（Kṣānti）—“承受、缓冲、化解”
构图语法（固定）
•结构：两股相向形体之间必须存在缓冲带/过渡层
•形体：更柔（圆角/弧面），避免锋利切面
•复杂度：形体 ≤ 3，渐变 ≤ 2
用色逻辑（从五色选）
•倾向“柔与清”：主色优先 白 / 蓝 / 绿 高明度
•对比色可选 红 或 黄（更轻、更小面积），避免刺激
蒙版规则（忍辱专属）
•Color Mask：双侧对向渐变（左→中、右→中）在中心融合
￮叠色 14–22%（中心最柔和）
•Opacity Mask：中心缓冲区透明度更高（更轻），两侧略实
￮变化幅度 ≤ 15%
•Blur Mask：中心融合处柔化 40–80px，形成“化开”的感觉
10 个几何变体家族
1.左右两块柔形对向靠近，中间留缝
2.上下两块对向靠近，中间留缝
3.两条弧线相向，中间形成柔光带
4.两块半透明薄片叠加形成缓冲层
5.双圆角矩形对向，但不碰撞
6.一实一虚对向（虚为半透明）
7.一条界线被柔化为过渡带
8.对向形体边缘渐隐（不碎裂）
9.中心仅一层薄雾式光晕（极克制）
10.两个小形体对向，中心为大留白

7.4 精进（Vīrya）—“推进、节律、明快”
构图语法（固定）
•结构：递进序列（阶梯/分段/节拍），方向明确但不符号化
•节奏：3–7 段，间距规律
•复杂度：形体 ≤ 4，线条 ≤ 2
用色逻辑（从五色选）
•倾向“明快而不躁”：主色优先 红 或 黄 的高明度版本
•对比色优先 蓝 或 绿（冷暖平衡）
•避免深红大面积压画面，保持“轻”
蒙版规则（精进专属）
•Color Mask：沿推进方向的线性渐变（左下→右上或相反）
￮叠色 12–18%
•Opacity Mask：推进末端更实（更有力），起点更透（更轻）
￮变化幅度 ≤ 12%
•Blur Mask：仅对推进序列的“尾部硬边”轻柔化 20–50px
10 个几何变体家族
1.3 段阶梯（最简）
2.5 段阶梯（均匀）
3.3 段块面递进（面积递增）
4.4 段块面递进（间距递减）
5.斜向分段带（不做箭头）
6.点状节拍（≤ 7 个点）沿斜线排列
7.细线 + 3 个节点（≤3）
8.递进块面 + 一条轻薄引导线
9.递进块面但轻微不对称（更自然）
10.两组递进序列交错但不叠复杂

7.5 禅定（Samādhi）—“收束、安住、澄寂”
构图语法（固定）
•结构：中心锚定（同心/环/集中）
•留白：六度中留白最大、元素最少
•复杂度：形体 ≤ 2，线条 ≤ 1（尽量做到）
用色逻辑（从五色选）
•倾向“清凉与净”：主色优先 白 / 蓝 / 绿 高明度
•对比色可选 黄（极少量作温度），避免红色主导
蒙版规则（禅定专属）
•Color Mask：中心柔和晕染（小范围径向），外部保持更干净
￮叠色 10–16%
•Opacity Mask：中心略实，外圈更透
￮变化幅度 ≤ 10%
•Blur Mask：外缘轻柔化 30–70px，形成“静气”但不糊
10 个几何变体家族
1.单一薄环（缺口可无）
2.单一圆角方（极简）
3.一点锚 + 一圈极淡环
4.两层同心（最多 2 层）
5.中心一块半透明薄片
6.中心一条极短线（≤1）
7.轻微偏心的中心形体（仍安住）
8.环 + 一块极淡色域（≤1）
9.中心形体边缘微光晕（非白硬光）
10.极简“空域”表现：主体更小，留白更大

7.6 般若（Prajñā）—“切透、澄明、洞见”
构图语法（固定）
•结构：清晰对照的两域 + 干净切面/切线（但保持轻，不做锐利刀痕）
•对比：强调“明度/冷暖”而非多色
•复杂度：形体 ≤ 3，线条 ≤ 2
用色逻辑（从五色选）
•倾向“澄明”：主色优先 白 / 蓝 / 黄 高明度
•对比色可选 绿 或 红（小面积点醒），确保不躁
蒙版规则（般若专属）
•Color Mask：分域渐变映射（左域/右域不同色相或明度）
￮叠色 12–20%
•Opacity Mask：切面附近更透（更“空”），两域更实
￮变化幅度 ≤ 12%
•Blur Mask：切线两侧做极轻柔化 20–40px，避免“硬切割”廉价感
10 个几何变体家族
1.一条斜切线分两域
2.一条竖切线分两域
3.切面为窄带（非线）分两域
4.两域之间留“极细空隙”
5.一块几何被切为两部分并错位
6.一实一虚两域对照（虚为半透明）
7.两域明度对照（同色相不同明度）
8.两域冷暖对照（两色相）
9.切面附近一层极淡光晕
10.分域但整体更轻（主体更小、留白更大）

7.8 生成执行文本（可直接作为统一出图指令）
•每一度：选定其“构图语法 + 蒙版规则 + 用色逻辑”
•从该度的 10 个变体家族中逐一生成：每个变体出 3 张（换色相组合与轻微布局偏移）
•每张必须满足：颜色 ≤ 3、形体 ≤ 4、线条 ≤ 3、无具象/无符号、留白 ≥ 50%
•导出：PNG（蒙版已应用）+ 源文件保留可调蒙版组

7.9 终审淘汰条件（任何一条命中即淘汰）
•出现任何可识别具象物/宗教器物/自然景观
•出现任何符号化元素（曼荼罗、法轮、禅圈、经文感图案、印章感图形等）
•颜色超过 3 色系或出现彩虹堆叠、荧光霓虹
•形体与线条过多导致“杂乱/拥挤/信息密度高”
•缩略图状态下糊成一团、发脏、发闷，不清澈不轻松
